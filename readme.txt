전체 목표는, 콘웨이의 생명게임을 디스플레이에 구동시키는 동시에, 게임의 상태를 확인하기 위해 LCD에 동적인 글자를 출력하고 살아있는 칸의 수를 계산해 7-segment에 표시하는 것입니다.

디스플레이
: 메모리 용량에 맞추어 16x8크기의 그리드에 콘웨이의 생명게임을 실행시켰습니다. 이를 화면에 출력되도록 했습니다. 디스플레이 위와 아래, 오른쪽과 왼쪽을 이어지도록 처리해 화면 가장자리에 의해 영향을 받지 않도록 했습니다. 


LCD
: 게임이 실행되는 동안 "Game of life"라는 문자가 움직이며 출력되도록 했습니다. 이때 null 대신 CGRAM에 새로운 문자를 정의해 사용했습니다. 

버튼
: 토글로 사용해 게임을 멈추고 재실행할 수 있도록 했습니다. 게임을 일시정지하게되면, LCD의 글자는 더이상 움직이지 않고, 생명게임의 세대별 업데이트가 없어 게임이 정지상태를 유지하게 됩니다. 

7-segment
: 살아있는 칸의 수를 출력하도록 했습니다. 최대 128개의 세포가 살아있을 수 있지만 일반적으로 두자릿수 정도의 세포가 살아있기에, 두자리의 세그먼트만 점등했습니다.
